# Product Overview

SWELLテーマの子テーマ - WordPressサイトのカスタマイズとブランディングのためのテーマ拡張

## Core Capabilities

- **テーマカスタマイズ**: SWELLテーマをベースとしたデザイン・機能拡張
- **モジュラー構造**: parts/配下での部分テンプレートによる段階的カスタマイズ
- **アセット管理**: Vite統合による効率的なCSS/JavaScript開発
- **レスポンシブデザイン**: モバイルファーストなレイアウトとインタラクション
- **コンテンツ配信**: ブログ、メインビジュアル、フッター要素の統一感あるプレゼンテーション

## Target Use Cases

- **ウェブサイトのブランディング**: 企業・個人サイトの独自デザイン実装
- **メインビジュアルカスタマイズ**: トップページの印象的なビジュアル演出
- **コンテンツレイアウト調整**: 投稿一覧、記事詳細、ナビゲーションの最適化
- **パフォーマンス最適化**: CSS/JSビルドプロセスによる高速化

## Value Proposition

- **SWELLエコシステム活用**: 人気テーマの機能を損なわずに拡張
- **モダン開発環境**: ViteとSCSSによる効率的なフロントエンド開発
- **WordPressベストプラクティス**: 子テーマ方式による安全なカスタマイズ
- **保守性**: 部分テンプレート方式による段階的な機能追加・修正
